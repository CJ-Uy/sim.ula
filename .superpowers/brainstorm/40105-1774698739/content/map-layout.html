Key Interactions

  • Toggle layers via the control panel (top-left of map)
  • Hover on map → tooltip shows vulnerability data for that point
  • Map stays full-height (viewport height minus header), form scrolls independently
  • On mobile: stacks vertically — map on top (40vh), form below
